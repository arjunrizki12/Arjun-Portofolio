# Arjun-Portofolio
Portfolio website pribadi Arjun Rizki — UI/UX Designer & Web Developer, mahasiswa Teknologi Informasi Universitas Brawijaya.

Website ini dibuat sebagai portofolio digital untuk menampilkan identitas profesional, latar belakang pendidikan, pengalaman organisasi, dan hasil karya Arjun Rizki di bidang UI/UX Design dan Web Development. Web ini dibangun dengan HTML, CSS, dan JavaScript

Fitur Fitur yang tersedia:
1. Responsive Design, tampil optimal di berbagai device seperti desktop, tablet dan mobile
2. Smooth Scroll Navigation
3. Link Navbar aktif otomatis saat scroll
4. Scroll Reveal Animation
5. Navigasi mobile dengan toggle menu (Hamburger)
6. Fallback image, yang mana placeholder otomatis jika gambar gagal dimuat

Sections
1. Home : Hero dengan foto profil dan tagline
2. About : Bio, spesialisasi, dan daftar skill
3. Education : Riwayat pendidikan (Universitas Brawijaya & SMAN 1 Sumberrejo)
4. Experience : Timeline pengalaman organisasi dan sertifikasi
5. Portfolio : Showcase project Study Space & Website Portfolio
6. Contact : Link ke Instagram, LinkedIn, dan WhatsApp
